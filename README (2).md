# Phishing Email Detection Model

A machine learning project using Scikit-learn to detect phishing emails.

## Features
- Detect phishing emails
- Analyze text features
- Display accuracy
- Generate confusion matrix

## Technologies Used
- Python
- Pandas
- Scikit-learn

## How to Run

pip install pandas scikit-learn

python phishing_detector.py

## Output
The model classifies emails as phishing or safe and generates a report.
